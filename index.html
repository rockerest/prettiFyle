PrettiFyle

a jQuery plugin to fix input[type="file"]

Download .zip Download .tar.gz View on GitHub

Regular HTML is all that's necessary: 

                
<input type="file" id="styled" />

PrettiFyle inputs with just: 

                
<script src="javascripts/jquery.prettiFyle.js">
<script type="text/javascript">
  $(function(){
    $( "#styled" ).prettiFyle();
  });
</script>

Then you can style the input and the button (CSS3Buttons used here): 

                
<script type="text/javascript">
  $(function(){
    $( ".prettiFyle-button" )
          .addClass( "positive pill" );
  });
</script>
